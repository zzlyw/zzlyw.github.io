My Project

My Recent Projects , Just "Click" on the pcitures to know More.

Vision-Tangible Physics-Based Interaction for Mixed Reality

In this project, we plan to construct a system based on the optical see-through head-mounted display. Our goal is to implement the most natural virtual-real interaction in the mixed reality environment.
Currently, we have built the basic platform, and the calibration for hand-eye coordination is also completed. Next, we want to improve the physics-based interaction to deformable objects, such as liquids and cloth.

Editable Reality

In this project, we are constructing a VR system to demonstrate the editable reality. In the proposed environment, we can generate Bezier Volume to deform any virtual object in order to make the deformation smooth. This project is focused on the visualization of a form of VR and a kind of direct interaction between the human and the virtual object. Users can make a customized design inside this environment.

VR Glove for Virtual Grasping

In this project, we are constructing a VR glove system to grasp virtual objects in the virtual environment. The hardware and the software are developed by us. Some simple applications using this VR glove is done.

Mirrored World Based on Self-Driven Virtual Agent

This is only a research plan. We need to create a virtual agent which can learn by itself. Then it has the essential features of a human.
The equivalent interaction between the virtual and the physical world is to be studied.
